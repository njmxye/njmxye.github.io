day02 编程进阶

1.知识回顾

  • 关于编程，学一门外语。

  • 解释器（CPython解释器）

  • 环境变量

  • IDE，集成开发环境。

    • vscode
    • pycharm，初学者推荐。
      • 社区版，主要用。
      • 专业版（2022.1版本激活问题、2021.3）

  • 编码

  • 输入和输出

  • 数据类型

  • 条件语句

  • 循环语句

  • 注释

    注意：快捷键 Ctrl + ?

2.函数

  • 面向过程编码，按照功能逐一实现。

  • 函数式编程，给一堆代码起个名字，后续根据这个名字找到函数并执行函数。

基于函数式编程来进行开发，优势：

  • 增加代码的重用性

  • 增加代码的可读性

2.1 函数的定义和执行

2.2 函数的参数

案例：发邮件

  • 第1步：注册邮箱。

  • 第2步：开启POP3/SMTP服务。

  • 第3步：记住授权码

  • 第4步：获取SMTP服务器

  • 通过代码发送

需求：

  • 定义一个函数，函数接受2个参数：目标邮箱、邮件内容。调用函数实现发送 邮件 。

  • 用户输入 + 发送邮件

3.再看数据类型

3.1 字符串 str

所有字符串都可以具备的功能。

  • 变大小写

    应用场景：

    • 网站上看到的验证码，内部不区分大小写，就是基于lower、upper实现。

    • 停止搜索

  • 字符串切割

    应用场景：

    • 请让用户输入一段文本，要求的格式，通过 , 将元素分割。

    • 现在有一个 csv 格式的文件，读取文件的内容。

    • 读取文件内容 + 读取到第一行的数据。

    • 读取文件内容+每一行

    • 读取文件 + 发送邮件工资条（非详细版本）

  • 去除空白

  • 是否以什么开头、是否以什么结尾，是否包含？

    应用场景：

    • 展示一个目录下的所有文件

    • 展示一个目录下的所有文件 + 寻找.mp4文件。

    • 展示一个目录下的所有文件 + 判断文件名中是否包含工资。

3.2 列表类型 list

只有列表才具备的个功能：

  • 追加

  • 插入

  • 删除

    应用场景：

    • 用户注册案例

    • 用户注册案例 + 判断是否用户姓 李，插队。排在签名。

  • 关于字符串的拼接

    • 字符串拼接

    • 字符串的格式化

    • 特殊的字符串拼接（单一）

    应用场景：

    • 让用户输入用户名和密码做用户注册。

    • 写文件，往文件中写内容。

    • 用户注册案例 + 写入文件

3.3 字典类型 dict

只有字典才具有的功能：

  • 根据键获取值

  • 循环相关

3.4 嵌套

  • 提取第一个手机号

  • 输入每个人的信息

案例：写文件数据

  • 获取网络数据

  • 获取网络数据 + 保存到csv格式的文件

  • 操作Excel

  • 获取网络数据 + 保存到 excel 文件中。
