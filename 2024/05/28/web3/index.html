Web3.0笔记

发表于2024-05-28|更新于2024-05-28
|阅读量:

Web3.0，也被称为Web 3.0，是关于全球信息网发展的一个概念，主要与基于区块链的去中心化、加密货币以及非同质化代币有关1。让我们深入了解一下这个概念。

Web3.0的核心思想是打破传统互联网的中心化模式，将权力归还给个人而非公司。让我们回顾一下互联网的发展历程：

发展历程

1. Web 1.0：只读 (1990-2004)

Web 1.0是指早期的互联网，主要由公司拥有的静态网站构成。用户之间的互动几乎为零，因此被称为“只读网络”。
这个阶段的互联网是由少数中心化实体控制和拥有的。

2. Web 2.0：能读能写（2004年至今）

随着社交媒体的兴起，Web 2.0时期于2004年开始。互联网不再是只读的，而是变成了读写网络。
用户不仅可以消费内容，还可以创作内容，并与其他用户互动。
少数互联网巨头开始掌控网络上的海量流量和价值。

了解更多: gitee

3. Web3.0：能读 - 能写 - 能拥有

Web3.0是在2014年以太坊推出后不久提出的概念。
Web3.0的核心思想是通过区块链、加密货币和非同质化代币将权力以所有权的形式归还给用户。
Web3.0具有以下特点：
去中心化：不再由中心化实体控制和拥有，而是由构建者和用户分配所有权。
无需许可：每个人都有平等参与的权限，没有人被排除在外。
原生支付功能：使用加密货币进行线上消费和汇款，不依赖传统银行或第三方支付机构。
去信任：通过激励措施和经济机制运作，不依赖受信任的第三方。

了解更多: gitee

为什么Web3.0很重要？

所有权：Web3.0赋予你数字资产的前所未有的所有权。例如，你可以直接拥有游戏内物品，而不受游戏创建者的限制。
抗审查：Web3.0允许你的数据位于区块链上，你可以在不同平台之间自由迁移。
去中心化自治组织（DAO）：在Web3.0中，你可以作为平台的一员拥有代币，参与平台的决策。
