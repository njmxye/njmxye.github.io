📖 使用帮助

导入题库：支持 Markdown (.md) 格式（如 exam.md）或 JSON 格式。

答题：单选题点击选项直接作答；多选题点击选项后点击「确认选择」；填空/简答/论述点击「显示答案」查看解析。

学习管理： ✅ 标记已掌握   ❌ 标记待复习   ⭐ 收藏题目

筛选：按题型、掌握状态筛选，支持关键词搜索。

数据持久化：所有学习进度自动保存到浏览器本地存储。
导出数据：可将学习数据导出为 JSON 文件备份。
